# 장바구니 프로젝트

## 프로젝트 개요

- **목표**: 사용자 중심의 직관적인 장바구니 시스템 구축 및 실시간 금액 계산 로직 구현 
- **주요 기능**: 
  - 상품 수량 증감 및 실시간 총액 업데이트
  - 10만원 이상 구매 시 배송비 무료 정책 적용 
  - 장바구니에서 결제 모듈(기존 개발건)로의 매끄러운 연동
- **참여 팀원**: 안영진

## 개발 환경

- **프레임워크**: React 18.x
- **사용 언어**: JavaScript (ES6+), CSS3
- **배포**: Vercel ([https://shooking-kappa.vercel.app/](https://shooking-kappa.vercel.app/))

## 설치 및 실행 방법

```bash
npm install
npm start
```

## 컴포넌트 구조

- **App.js**: 전체 페이지 라우팅 및 장바구니 데이터 상태 관리
- **Cart.js**: 수량 조절, 배송비 계산 등 장바구니의 핵심 로직 담당
- **Cart.css**: 시안 기반의 블랙 & 화이트 톤 레이아웃 및 컴포넌트 스타일링

## 테스트 방법

- **단위 테스트**: npm test
- **컴포넌트 문서화**: npm run storybook 실행 후 Button.mdx, Cart.mdx 확인
- **기능 테스트**: 배포 URL 접속 후 10만원 기준 배송비 변동 여부 확인

## 유의사항 및 알려진 이슈

- **뒤로가기 버튼**: 장바구니 상단의 뒤로가기(←) 버튼 클릭 시 이전 상품 목록 페이지로 정상 이동합니다.
- **결제 연동**: '결제하기' 버튼 클릭 시 기존에 개발된 '보유 카드' 화면으로 데이터가 연동됩니다.
