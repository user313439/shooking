# 상품 목록 페이지 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명: 슈킹 쇼핑몰 상품 목록 페이지 (모바일)

### 사용하게 될 기술:  React

### 문서 작성자: 안영진

## 📝 고객 요구사항 정리
* 20-30대 젊은 세대를 타겟으로 하며, 신발을 주력 상품으로 판매함.
* 모바일 환경에서도 편리하게 이용할 수 있도록 최적화된 웹페이지 구축.
* 고해상도 신발 사진을 강조하여 시각적으로 매력적인 목록 제공.
* 장바구니 담기 시 실시간으로 상품 갯수를 확인할 수 있는 직관적 UI 제공.
* 디자인 시안

## 📋 기능 목록
* **상품 리스트**
  * 상품을 2개의 컬럼으로 구성된 그리드 레이아웃으로 보여줌.
  * `<ProductList />`
* **상품 카드**
  * 고해상도 이미지가 포함된 상품 카드 구성.
  * '담기' 버튼 클릭 시 장바구니에 추가되며, 버튼 상태가 변화함.
  * `<ProductCard />`
* **헤더 및 장바구니 상태**
  * 장바구니에 상품을 담은 후, 헤더에 담긴 상품의 총 갯수가 실시간으로 업데이트됨.
  * `<Header />`
* **기타**
  * 작업 결과물을 실시간으로 확인할 수 있는 테스트(배포) URL 제공 예정.
