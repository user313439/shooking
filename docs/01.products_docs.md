# 상품 목록 페이지 프로젝트

## 프로젝트 개요

* **프로젝트 목표**: 2030 세대를 타겟으로 하는 신발 전문 쇼핑몰의 모바일 최적화 상품 목록 페이지 구현
* **주요 기능**: 고해상도 신발 이미지 강조, 2열 그리드 레이아웃, 실시간 장바구니 담기 및 수량 업데이트 기능
* **참여 팀원**: 안영진

## 개발 환경

* **프레임워크 및 라이브러리**: React 18, JavaScript ES6+
* **상태 관리**: React Hooks (useState)
* **스타일링**: CSS3 (Flexbox & Grid 사용)

## 설치 및 실행 방법

프로젝트를 로컬에서 실행하기 위한 설치 방법과 명령어

```
npm install
npm start
```

## 컴포넌트 구조

주요 컴포넌트들의 구조와 역할

예시:

**ProductList**: 전체 상품 데이터를 관리하고, 2개의 컬럼 그리드로 상품들을 배치하는 컨테이너입니다.

**ProductCard**: 각 상품의 이미지, 브랜드명, 가격 정보를 표시하며 '담기' 버튼 클릭 시 장바구니에 추가되는 기능을 처리합니다.

**Header**: 쇼핑몰 로고와 장바구니에 담긴 상품의 총 갯수를 실시간으로 업데이트하여 보여줍니다.

## 테스트 방법

테스트 케이스 및 실행 방법


**테스트 케이스**: 상품 카드 렌더링 확인, 장바구니 버튼 클릭 시 수량 증가 로직 검증

**실행 방법**: npm test


## 유의사항 및 알려진 이슈

**성능**: 고해상도 이미지 사용으로 인해 네트워크 속도에 따라 로딩 지연이 발생할 수 있습니다.

**레이아웃**: 현재 모바일 환경에 최적화되어 설계되었습니다.
