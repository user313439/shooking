# 장바구니 페이지 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명: 
슈킹 온라인 장바구니 페이지 구축

### 사용하게 될 기술: 
- React 18.x
- JavaScript (ES6+)
- CSS3 (모바일 최적화 레이아웃)

### 문서 작성자: 
안영진

## 📝 고객 요구사항 정리
1. **구매 흐름 변경**: 상품 상세 페이지에서 바로 결제하던 방식에서 '장바구니'를 거쳐 결제하는 방식으로 변경
2. **비교 환경 제공**: 여러 상품을 담아 한눈에 가격과 수량을 비교할 수 있는 UI 구현
3. **금액 정확성**: 상품별 합계 금액 및 최종 결제 금액의 정확한 실시간 계산
4. **배송비 정책**: 총 주문 금액이 100,000원 이상일 경우 배송비 0원 처리 (미만 시 3,000원 발생)
5. **테스트 확인**: 배포 후 실제 동작을 확인할 수 있는 테스트 URL 제공
6. **디자인 시안**: https://www.figma.com/design/DKeR5wR65RkxcAWReoBccl/%EC%9D%91%EC%9A%A9%EC%86%8C%ED%94%84%ED%8A%B8%EC%9B%A8%EC%96%B4-%EB%94%94%EC%9E%90%EC%9D%B8-%EC%8B%9C%EC%95%88?node-id=1-131&p=f&t=MtKs51q8EppqgN1h-0

## 📋 기능 목록
- **장바구니 리스트**: 담긴 상품의 브랜드, 상품명, 가격, 이미지 표시
- **수량 조절 기능**: 버튼(`+`, `-`)을 통한 실시간 수량 변경 및 개별 합계 금액 업데이트
- **상품 삭제**: 장바구니에 담긴 개별 상품 삭제 기능
- **결제 예상 금액 산출**:
    - 총 상품 가격 계산
    - 배송비 조건부 계산 (10만원 기준)
    - 최종 결제 총액 표시
- **네비게이션**: 
    - 상단 '뒤로가기'를 통한 이전 화면 이동
    - 하단 '주문하기' 버튼 클릭 시 기존 결제 모듈(보유 카드 화면) 연동
- **장바구니 비우기**: 결제 완료 시 장바구니 내역 초기화 로직
