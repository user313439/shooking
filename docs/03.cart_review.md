# 장바구니 페이지 리뷰

## 고객사 요구사항
> 이메일로 요청하신 핵심 기능과 피그마 시안의 세부 디자인 요소를 모두 반영했습니다.

- **금액 계산**: 담긴 상품의 개수만큼 가격이 정확하게 합산되어야 함
- **배송비 정책**: 10만원 이상 구매 시 배송비가 '무료'로 자동 전환되어야 함(10만원 이하시 3000원)
- **수량 조절**: +, - 버튼을 통해 페이지 새로고침 없이 수량을 즉시 변경할 수 있어야 함
- **뒤로가기**: 쇼핑을 계속할 수 있도록 상단에 메인 목록으로 돌아가는 버튼 배치
- **결제 연동**: '결제하기' 버튼 클릭 시, 이미 등록된 카드 목록을 확인하고 결제할 수 있는 화면으로 연결

## 요구사항에 따른 기능 동작 확인
> 실제 화면에서 구현된 세부 기능들을 확인한 결과입니다.

- **배송비 무료 혜택 자동 적용**: 장바구니에 담긴 상품들의 합계가 10만원이 넘는 순간, 배송비 3,000원이 '무료'라는 문구로 바뀌며 총 결제 금액에서 제외되는 것을 확인했습니다.
- **직관적인 수량 관리**: 각 상품 옆의 버튼을 누르면 숫자가 변함과 동시에 하단의 '상품 금액'도 실시간으로 다시 계산되어 고객이 혼동하지 않도록 구현했습니다.
- **끊김 없는 구매 흐름**: 장바구니에서 최종 확인을 마친 후 '결제하기' 버튼을 누르면, 이전에 개발 완료된 '카드 선택' 창이 모달 형태로 바로 나타나 즉시 결제가 가능합니다.
- **모바일 최적화 레이아웃**: 시안에서 요구하신 검은색 상단 바와 깔끔한 흰색 배경을 그대로 재현하여, 스마트폰 환경에서도 상품 정보가 한눈에 들어오도록 디자인했습니다.
- **이전 화면 복귀**: 좌측 상단의 화살표를 누르면 장바구니에 담은 내역을 유지한 채로 신발 목록 페이지로 안전하게 돌아갑니다.
