# 장바구니 페이지 (Cart Page)

사용자가 선택한 상품을 확인하고 최종 결제 금액을 산출하는 핵심 페이지입니다.

## 1. 주요 구성 요소
- **장바구니 리스트**: 담긴 상품의 이미지, 브랜드, 가격 정보를 나열합니다.
- **수량 조절기**: `+`, `-` 버튼을 통해 상품 개수를 변경하며, 실시간으로 금액에 반영됩니다.
- **결제 요약**: 상품 합계와 배송비를 계산하여 최종 금액을 노출합니다.

## 2. 배송비 정책 (Business Logic)
- 총 상품 금액 **10만원 미만**: 배송비 3,000원 발생
- 총 상품 금액 **10만원 이상**: 배송비 0원 (무료 배송)
